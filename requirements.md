# All the packages that are needed to be installed

All the installs will be done in the Anaconda PowerShell.

### OpenCV
```python
$ pip install opencv-python
```

### Tensorflow
```python 
$ pip install tensorflow
```

### Pandas
```python
$ pip install pandas
```

### Matplotlib
```python
$ pip install matplotlib
```

### NumPy
```python
$ pip install numpy
```

### Scikit-learn

```python
pip install scikit-learn
```

### Pillow
