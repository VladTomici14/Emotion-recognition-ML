# All the packages that are needed to be installed

All the installs will be done in the Anaconda PowerShell or in the Linux terminal.

### OpenCV (4.5.3)
```shell script
pip install opencv-python
```

### Tensorflow (2.6.0)
```shell script
pip install tensorflow
```

### Pandas (1.1.5)
```shell script
pip install pandas
```

### Matplotlib (3.2.2)
```shell script
pip install matplotlib
```

### NumPy (1.19.5)
```shell script
pip install numpy
```

### Scikit-learn (0.23.1)
```shell script
pip install scikit-learn
```

### Pillow (8.3.2)
```shell script
pip install Pillow
```

### Argparse (1.1)
```shell script
pip install argparse
```
